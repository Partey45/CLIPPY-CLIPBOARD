# 📋 Clippy — Smart Clipboard Manager

Clippy is a Windows clipboard manager that lets you organize, search, and 
paste your clipboard history using grouped entries and hotkeys.

## ✨ Features

- 📁 **Group-based organization** — Organize clipboard entries into named groups
- 🔀 **Variations support** — Store multiple variations under a single group
- 🔍 **Instant search** — Quickly find any saved entry
- ⌨️ **Hotkey support** — Trigger Clippy without leaving your keyboard
- 🖥️ **Lightweight** — Runs quietly in the background on Windows

## 🚀 Getting Started

### Run from source
pip install -r requirements.txt
python clippy.py

### Build executable
build_exe.bat
